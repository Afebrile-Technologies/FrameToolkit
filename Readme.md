# FrameToolkit.AspNetCore

A comprehensive utility library for ASP.NET Core development that provides essential extensions and abstractions to accelerate your projects.

## Features

### 🛠️ Extensions
- **String Extensions**: Common string manipulation utilities
- **LINQ Extensions**: Enhanced query operations
- **IEnumerable Extensions**: Collection utilities and helpers

### 🏗️ Architecture Abstractions
- **CQRS Pattern**: Interfaces and base handlers for Command and Query Responsibility Segregation
- **Domain Event Handler**: Custom event handling for domain-driven design
- **Minimal APIs Wrapper**: Simplified setup and configuration for Minimal APIs

### 🔧 Core Utilities
- Various useful abstractions for building clean, maintainable ASP.NET Core applications

## Installation

```bash
dotnet add package FrameToolkit.AspNetCore
